{\bf Questão 47} 

O resto da divisão do polinômio $p(x)$ por $x-1$ é igual a $5$. Dividindo o mesmo polinômio
 por $x+1$, obtém-se o mesmo resto: $5$. Dividindo o polinômio $p(x)$ por $(x-1)\cdot (x+1)$, o resto obtido é:
\begin{multicols}{2}
\begin{enumerate}
		\item ( ) $0$.
		\item ( ) $5$.
		\item ( ) $25$.
		\item ( ) $125$.
		\end{enumerate}
\end{multicols}
