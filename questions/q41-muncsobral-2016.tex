{\bf Questão 41}
 Um triângulo equilátero $ABC$ tem área $S$. Ao tomarmos os pontos médios de cada lado, obtemos um novo
triângulo $EFG$. A relação entre a área de $EFG$ e $ABC$ é:

\begin{multicols}{4}
\begin{enumerate}
		\item () $\frac{1}{4}$ 
		\item () $\frac{1}{2}$ 
		\item () $2$ 
		\item () $4$ 
\end{enumerate}
\end{multicols}
